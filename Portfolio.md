# 포트폴리오 철학 및 운용 원칙

> 마지막 업데이트: 2026-05-07

---

## 1. 핵심 철학

**"레짐을 읽고, 계좌 구조를 활용하며, 분산의 실질적 효과를 극대화한다."**

이 포트폴리오는 세 가지 원칙을 동시에 추구한다.

1. **동적 자산 배분** — 시장 국면(레짐)에 따라 비중을 자동으로 조정한다. 고정 비중은 특정 국면에서 필연적으로 과대·과소 노출된다.
2. **계좌 효율성** — KRW와 USD 계좌의 예산 한계를 인식하고, 각 통화의 예산을 가장 효율적인 자산에 배분한다.
3. **진정한 분산** — 주식-채권 상관에 그치지 않고, 상관관계가 낮은 원자재·managed futures를 추가해 포트폴리오의 실질 분산 효과를 높인다.

---

## 2. 자산군 구성 (4축 포트폴리오)

| 축 | 자산군 | 역할 |
|----|--------|------|
| 1 | Equity ETF (KODEX) | 장기 성장 엔진 — KRW 계좌 효율 활용 |
| 1 | Equity Factor (VTV) | Value 팩터 분산 — USD 계좌 |
| 1 | Equity Individual (TSLA·PLTR) | 고베타 성장주 — 레짐 무관하게 일정 비중 유지 |
| 2 | Bond + Gold | 리스크-오프 시 손실 완충, 인플레이션 헤지 |
| 3 | Commodities (DBC) | 인플레이션 베타 확보, 에너지·금속·농산물 분산 |
| 4 | Managed Futures (DBMF) | 추세추종 CTA 복제, 주식·채권 모두와 낮은 상관 |

**Commodities와 Managed Futures를 추가한 이유:**  
전통적인 주식-채권 포트폴리오는 인플레이션 상승·스태그플레이션 국면에서 동시에 하락하는 구조적 취약점이 있다. 원자재는 인플레이션 국면에서 양(+)의 베타를, Managed Futures는 방향에 무관하게 추세에서 수익을 추구하므로 이 두 자산을 추가하면 어떤 국면에서도 최소한 하나의 축이 작동한다.

## 3. 동적 비중 계산 (KRW/USD 비율 반응)

목표 비중은 **전체 포트폴리오 기준 자산군 퍼센트**로 정의되어 있다. 매 실행 시 실제 계좌 잔고를 읽어 per-account 비중을 런타임에 계산한다.

### USD 배정 우선순위

USD 예산이 목표에 미치지 못할 때 자동으로 조정된다.

```
1순위 — commodity (DBC) + managed_futures (DBMF)  → 전액 보장
2순위 — equity_factor (VTV) + equity_individual (TSLA·PLTR)  → 예산 초과 시 비례 축소
3순위 — bond_usd (IEF·SHY)                        → 잔여 예산에만
잔여 USD 있으면 → 1~3 항목 비례 확대 (USD 계좌 100% 소진)
```

### 비율 변화에 따른 자동 조정

Risk-On 레짐, 전체 200M 기준:

| KRW:USD | Equity | Commodity | Managed F. | 특이사항 |
|---------|--------|-----------|------------|---------|
| 14:6 | 59% | 10% | 8% | TSLA+PLTR 목표 11% → ~6% (USD 한도) |
| 12:8 | 62% | 11% | 9% | TSLA+PLTR 완전 배정, CM+MF 자연 증가 |
| 10:10 | 60% | 14% | 11% | CM+MF 비중 최대화 |

**핵심:** USD가 커질수록 대체자산(CM+MF) 비중이 자동으로 늘고, 줄어든 KRW만큼 equity_etf(KODEX)가 흡수한다. config 수정 없이 비율 변화에 대응된다.

## 4. 계좌 구조와 자산 배치 원칙

### 핵심 원칙: "KRW 대체재 유무"

```
KRW 대체재 있음 → KRW 계좌에서 처리 (환전 불필요)
KRW 대체재 없음 → USD 계좌 전용
```

| 자산 | 계좌 | 이유 |
|------|------|------|
| KODEX S&P500 (379800) | KRW | SPY의 완전 대체재. USD 소모 불필요 |
| KODEX 나스닥100 (379810) | KRW | QQQ의 완전 대체재. USD 소모 불필요 |
| TIGER 미국채10년 (305080) | KRW | IEF·TLT의 KRW 대체재 |
| ACE KRX금현물 (411060) | KRW | 금 현물 연동, KRW로 거래 가능 |
| SOL 초단기채권 (469830) | KRW | 유동성 버퍼, T+2 결제 대응 |
| **VTV** | **USD** | Value 팩터 ETF, KRW 대체재 없음 |
| **TSLA, PLTR** | **USD** | KRW 대체재 없는 개별주 |
| **DBC** | **USD** | KRW 시장에 동등 원자재 바스켓 ETF 없음 |
| **DBMF** | **USD** | KRW 시장에 CTA 복제 ETF 없음 |
| IEF, SHY | USD | 방어 레짐에서 USD 계좌 내 채권 비중 |

### 계좌 구조의 실질 효과

자산 규모 예시: KRW 계좌 140M + USD 계좌 ~$43K(약 60M KRW 환산)

- USD 예산이 30%로 제한적이므로, USD를 SPY/QQQ에 사용하면 TSLA·PLTR·DBC·DBMF 중 일부는 매수 불가
- KRW로 KODEX를 활용하면 USD 예산 전체를 "KRW 대체재 없는 자산"에 집중할 수 있음
- 결과적으로 USD 계좌는 항상 100% 소진되며 예산 낭비가 없음

## 5. 레짐 기반 동적 배분

### 레짐 정의

시스템은 4개의 시장 국면(레짐)을 인식한다.

| 레짐 | 시장 특성 | 전략 |
|------|-----------|------|
| **Risk-On** | 모멘텀 강세, 변동성 낮음, 크레딧 스프레드 안정 | 주식 비중 최대화 |
| **Neutral** | 혼조세, 방향성 불명확 | 균형 유지 |
| **Risk-Off** | 모멘텀 약세, 채권 강세 | 방어 자산 비중 확대 |
| **High-Vol** | 변동성 급등, VIX 상승 | 현금·채권 최대화 |

레짐 감지는 규칙 기반(모멘텀·VIX·크레딧 신호)과 HMM(Hidden Markov Model) 앙상블로 작동하며, 신뢰도 미달 시 Neutral로 폴백, 히스테리시스 필터(3회 연속 확인 + 5일 쿨다운)로 잦은 전환을 방지한다.

실제 비중은 HMM 사후 확률로 레짐 간 연속 블렌딩(Continuous Exposure)이 적용된다. 아래 수치는 각 레짐 확률이 100%일 때의 순수 배분 기준이다.

## 6. Equity 비중에 대한 입장

이 포트폴리오는 **의도적으로 equity 비중이 높다.** 추천 안(equity 30~40%)보다 공격적인 비중을 선택한 이유:

- 장기 자산 증식에서 equity의 기대수익이 여전히 가장 높다
- TSLA·PLTR은 레짐에 무관하게 일정 수준을 유지해 성장 베팅을 끊지 않는다
- Managed Futures와 Commodities는 equity 하락 국면에서 보완 역할을 하므로, 동일한 리스크 수준에서 더 높은 equity 비중을 허용한다
- 레짐 전환 시 자동으로 equity 비중이 축소되므로, Risk-On에서 높은 비중을 유지해도 시스템적으로 방어된다

## 7. 종목 선택 이유

### TSLA (Tesla)
AI·자율주행·에너지 저장이 교차하는 기술주. 단순 자동차 회사가 아닌 에너지 플랫폼으로의 전환을 베팅한다. 높은 변동성을 감수하되 레짐 전환 시 비중을 자동 축소해 리스크를 관리한다.

### PLTR (Palantir)
AI 기반 데이터 분석 플랫폼. 국방·정부·민간 대형 고객 기반으로 구조적 성장이 기대된다. TSLA와 상관관계가 낮아 개별주 내 분산 효과도 있다.

### VTV (Vanguard Value ETF)
Value 팩터를 추구하는 광범위 미국 대형가치주 ETF. 성장주 중심의 KODEX 나스닥100·TSLA·PLTR과 팩터 분산 효과가 있으며, Risk-Off 국면에서 상대적으로 방어적이다.

### DBC (Invesco DB Commodity Index)
에너지·금속·농산물을 균형 있게 담은 광범위 원자재 바스켓 ETF. 단일 상품 리스크를 제거하면서 인플레이션 베타를 확보한다. 선물 롤오버 구조도 고려했을 때 장기 보유에 적합한 Broad Basket.

### DBMF (iMGP DBi Managed Futures Strategy)
기관 CTA(추세추종) 전략을 복제한 ETF. 주식이 오를 때도, 내릴 때도 추세 방향으로 포지션을 구성하므로 주식·채권과 낮은 상관관계를 유지한다. 포트폴리오의 "4번째 축"으로 어떤 레짐에서도 일부 역할을 수행한다.

## 8. 리스크 관리

### 드로우다운 기반 자동 스케일다운

채권·금·현금은 유지하고 equity만 단계적으로 축소한다. 바닥에서 전량 현금화하는 패턴을 방지하기 위함이다.

| 드로우다운 | 조치 |
|-----------|------|
| -10% ~ -20% | equity × 0.75 (75% 유지) |
| -20% ~ -30% | equity × 0.40 (40% 유지) |
| -30% 이상 | equity = 0 (채권·금·현금 유지) |

### 변동성 타겟팅

실현 변동성(연환산)이 목표(10%)를 초과할 때 equity 비중을 비례 축소한다.

```
scale = clip(target_vol / realized_vol, floor=0.65, 1.0)
equity × scale → 잉여분 cash로 이동
```

레버리지 없음: realized_vol < target_vol이면 scale = 1.0(축소 없음).

### T+2 결제 지연 대응

KIS(한국투자증권)의 T+2 결제 구조로 인해 매도 대금이 즉시 가용하지 않은 문제를 두 가지 방식으로 해결한다.

- **Pre-Funding Buffer**: SOL 초단기채권(469830)을 KRW 계좌의 최소 7%로 유지. 매도 결제를 기다리지 않고 이 버퍼로 먼저 매수를 집행한다.
- **Synthetic Exposure**: USD 매수가 결제 지연으로 실행되지 못할 경우, KRW 동등 자산(KODEX, TIGER채권 등)을 임시로 매수해 "노출 공백"을 제거한다. 다음 실행 사이클에서 USD 매수가 성공하면 자동으로 정리된다.

### 레짐 전환 히스테리시스

- 동일 레짐이 3회 연속 감지되어야 전환 확정 (노이즈 필터)
- 마지막 전환 후 최소 5일 쿨다운 (과도한 거래 방지)
- 레짐 신뢰도 40% 미달 시 Neutral 폴백

## 9. 이 포트폴리오가 하지 않는 것

- **개별 종목 발굴·리서치** — 종목 선택은 수동으로, 시스템은 비중 관리만 담당한다.
- **마켓 타이밍** — 레짐은 현재 상태를 기술할 뿐, 미래를 예측하지 않는다.
- **환율 헤지** — KRW/USD 환노출은 의도적으로 관리하지 않는다. 분산 효과의 일부로 수용한다.
- **레버리지** — 모든 포지션은 현금 매수 기준이다.
