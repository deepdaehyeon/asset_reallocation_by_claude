# 재실험: 고정 4지표 전환 후 우선순위 높음 5건

*작성일: 2026-06-12*
*상태: 완료 (5/5). 코드 변경 없음 — floor·drift 조정은 사용자 확인 대기.*

## 배경

CLAUDE.md 규칙4(2026-06-12)로 평가 기준을 **롤링 CAGR · Ulcer Index · 회복기간 ·
Martin Ratio** 4지표로 고정했다. 기존 실험들은 Sharpe/Calmar/MaxDD 기준으로 결론을
냈으므로, 결론이 지표에 직접 의존하는 "높음" 5건을 새 기준으로 재실행한다.

판정 원칙: **Martin(=CAGR/Ulcer)이 1차 판정**, 동률·근소차는 **회복기간·롤3y최악**으로
가른다. CAGR·MaxDD·Sharpe는 보조참고. (USD 단일통화·단일 포트폴리오 백테스트 —
라이브 USD합성 순환매 미반영.)

대상: ① vol_targeting_sweep ② voltarget_blend_ablation ③ drift_threshold_resweep
④ regime_value_endtoend ⑤ risk_hardening

### 핵심 발견 (옛 결론과 충돌)

1. **floor를 조일수록 Martin 하락 + 최장 underwater 증가.** floor 0.80→0.35로 조이면
   Martin 2.55→2.34, 최장UW 409→523일. equity를 깊이 축소하면 반등 구간 참여가 줄어
   물려있는 기간이 길어진다(core_satellite의 core100 = 최장 UW와 동일 메커니즘).
2. **현행 floor 0.50은 Martin 2.43·UW 514일로 중하위.** 이 값은 2026-05-30 MaxDD 중심
   스윕에서 채택됐는데, 새 기준(회복기간 가중)에서는 floor 0.65~0.80이 우위다.
3. **tv_mult 축소(0.70)는 회복기간을 단축**(84일, 최단)하지만 롤3y최악도 같이 낮춘다(+3.7%).
   하락 고통은 줄지만 경로 수익도 깎이는 트레이드오프.
4. Martin은 전 구간 2.34~2.55로 평탄 — vol targeting의 가치는 Martin보다 **MaxDD(보조)와
   회복기간**에서 드러난다.

### 시사점 (코드 변경 보류, 사용자 판단 필요)

새 기준에서 **floor 0.50 → 0.65 상향**이 Martin·회복·UW 모두 개선(2.43→2.50, 110→106일,
514→437일)하며 MaxDD는 -15.1%→-15.9%로 0.8pp만 악화. 단, 이는 단일통화 백테스트 결과로
라이브 방어력 체감과 다를 수 있어 **변경은 별도 확인 후 결정**한다.

### 핵심 발견

1. **vol targeting은 새 기준에서 net-negative로 보인다.** 끄면 Martin +0.57, 회복 −95일,
   최장UW 714→523일. 단 MaxDD −10.9%→−14.4%, COVID −5pp·Bear22 −3pp로 **위기 깊이 방어는
   진짜다**. = vt는 *하락 깊이*를 줄이는 대신 *회복 길이*와 CAGR를 희생. 옛 MaxDD 중심
   시각이 놓친 비용을 회복기간 지표가 드러냄.
2. **blend는 가치 유지.** both_off(blend off)가 Ulcer 4.96·Martin 1.72로 최악. blend는
   Ulcer를 낮춰(하락 고통 완화) 밥값을 한다 — [[project-voltarget-blend-defense-engine]]의
   blend 부분은 새 기준에서도 성립.
3. **메모리 보정 필요.** "voltarget+blend 둘 다 방어 엔진"은 MaxDD 기준 결론. 새 4지표에서
   **blend=방어 엔진(Ulcer↓), vol targeting=깊이-길이 트레이드오프**로 분해된다.

### core30과의 연결

이 ablation은 core_satellite 미적용. 라이브 core30은 30%를 정적 Goldilocks(vt 없음)로 묶어
회복 구간에 풀 참여 → vt의 회복기간 비용을 구조적으로 상쇄한다. 즉 core30 채택이
이 트레이드오프에 대한 이미-내린 답의 일부.

### 횡단 통찰 (새 기준이 드러낸 것)

옛 MaxDD 중심 시각에서는 보이지 않던 **회복기간 비용**이 일관되게 드러났다:
equity를 깊이/오래 축소하는 모든 장치(vol targeting floor↓, drawdown scaling, 정적 방어)는
*하락 깊이*는 줄이지만 *회복 길이(underwater)*를 늘린다. 본 포트폴리오의 1순위 목표가
하락회피이고 사용자가 장기보유자이므로, 회복기간을 직접 평가하는 새 4지표가 더 적합하다.
core30 채택은 이 비용에 대한 구조적 답(코어가 회복에 풀 참여)으로, 사후적으로 정당화된다.

### 후속 의사결정 대기 (코드 변경은 사용자 확인 후)

- **vol_targeting floor 0.50 → 0.65**: 4지표 우위(Martin 2.43→2.50, UW 514→437일), MaxDD
  −0.8pp만 악화. 단 단일통화 백테스트라 라이브 방어력 체감 확인 필요.
- **drift 1.5% → 1.0%**: 백테스트 Martin +0.15이나 라이브 회전 ~2배 증가 리스크. 보류 권장.
