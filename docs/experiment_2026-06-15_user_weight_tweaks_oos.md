# 사용자 지정 비중조정(골디·리플 가치주↑) 워크포워드 OOS

## SUMMARY
- **무엇을**: 사용자 지정 2건 — G[Goldilocks] equity_factor 5→7%·gold 10→8%, R[Reflation] equity_factor 8→10%·managed_futures 12→10% — 을 워크포워드 OOS(TRAIN 2010–2018 / TEST 2019–2025, 4지표)로 측정(`scripts/walkforward_weight_tweaks_oos.py`).
- **핵심 수치**: G는 학습창 ΔMartin −0.06·**검증창 −0.11**·MaxDD −14.9→−15.2%로 **악화**(gold→factor 교체가 분산 손실로 역효과). R은 학습 +0.01·검증 +0.01로 **중립**(엔진 흡수, 무해하나 노이즈 내).
- **채택 여부·결론**: **G 미채택**(OOS 손해 — gold는 단독 Martin −0.14라도 분산자라 빼면 손해), **R은 사용자 판단**(무해·방향 정합이나 이득은 노이즈 수준). 동시점 *단독* Martin이 약하다고 빼면 안 된다는 교훈을 OOS로 재확인([[feedback-regime-targets-no-tuning]]).

## 배경
사용자 "레짐별로 비중을 하나씩 고쳐나가자". 골디락스·리플레이션 가치주(VTV+AVUV=equity_factor,
60:40 유지) 비중을 올리고 재원을 약한 자산군(골디 gold, 리플 DBMF)에서 마련하는 2건.
동시점 진단상 둘 다 "약→강 교체"라 C1·C2(앞 실험)와 달리 OOS 생존 가능성을 기대.

## 해석
- **G = 역효과**: 학습·검증 모두 Martin 하락(OOS −0.11), MaxDD·Ulcer 악화. gold 단독 동시점
  Martin은 −0.14로 최악이지만, gold는 주식과 저상관(+0.09) **분산자**다. 이를 주식과 상관 높은
  equity_factor(주식상관 0.63)로 바꾸면 포트폴리오 집중도가 올라 변동성 타게팅이 위험을
  더 인식 → 위험조정수익 악화. **"단독 Martin 약함"이 "분산 기여 약함"을 뜻하지 않는다**는
  MV-vs-Martin 긴장([[experiment_2026-06-15_mv_optimizer_robust_candidates_oos]])의 OOS 실증.
- **R = 중립**: ΔMartin +0.01(양 구간). DBMF의 분산 가치(주식상관 ~0.21)가 2% 축소로는 거의
  안 깨지고, 가치주 추가도 엔진에 흡수 → 무해하나 이득도 노이즈 내. 방향은 데이터와 정합.
- C1·C2(앞 실험)에 이어 **세 번째로** per-regime 비중 미세조정이 OOS에서 엔진을 못 이김.
  단독수익이 강한 신호처럼 보여도(가치주 Martin 13.98) 파이프라인이 흡수하거나, 분산 손실로 역전.

## 결론·채택
- **G(골디 gold→factor): 미채택.** OOS 손해. gold 10%는 단독 약체가 아니라 분산 보험으로 유지.
- **R(리플 factor↑/DBMF↓): 사용자 판단.** 무해·방향 정합이나 4지표 이득은 노이즈 수준 →
  데이터 근거로 "해야 한다"고 말할 수 없음. 라이브 미반영(확인 전까지).
