# 레짐 비중 로버스트니스 검증 보고서

**검증 일시**: 2026-05-07  
**백테스트 기간**: 2010-01-01 ~ 2024-12-31 (15년)  
**리밸런싱**: 주별(금요일) / 거래비용 0.1% 편도  
**벤치마크**: 60/40 (SPY 60% + IEF 40%)

---

## 검증 목적

레짐별로 설정한 비중이 **과적합(overfitting) 없이 의도대로 작동**하는지 확인.  
최고 수익률을 찾으려는 최적화가 아니라, 어느 시기에서도 의도한 공격/방어 역할을 수행하는지 검증.

---

### 2014-16 이상치 원인 분석

QE 강세장(2014-16) 기간에 전략 CAGR +0.9%, Sharpe -0.47로 유일하게 부진. 원인은 비중 설계 문제가 아닌 **데이터 한계**:

- PLTR: 2020-09-30 이전 데이터 없음 → equity_individual 비중 강제 재배분
- DBMF: 2019-05-10 이전 데이터 없음 → managed_futures 비중 강제 재배분
- KRW ETF(379800/379810): 초기 데이터 부재로 프록시 처리 불완전

나머지 4개 기간은 양수 Sharpe + DD 절감을 일관되게 달성.

## 2단계: 레짐 의도 달성 검증

> 각 레짐에서 설정한 비중이 의도한 역할(공격/방어)을 실제로 수행하는지.  
> **성장 레짐**: 해당 레짐 체류 기간에 CAGR > 0%  
> **방어 레짐**: 해당 레짐 체류 기간에 |전략 MaxDD| < |BM MaxDD|

| 레짐 | 의도 | 체류일 | 체류 비중 | 전략 CAGR | BM CAGR | 전략 MaxDD | BM MaxDD | 달성 |
|------|------|------:|--------:|--------:|-------:|---------:|--------:|------|
| Goldilocks | 성장추종 | 2,429 | 64% | +12.6% | +8.1% | -12.3% | -10.0% | ✓ OK |
| Reflation | 성장추종 | 100 | 3% | +15.0% | +33.9% | -5.8% | -4.8% | ✓ OK |
| Slowdown | 손실방어 | 915 | 24% | +2.6% | +1.7% | -14.3% | -30.2% | ✓ OK |
| Stagflation | 손실방어 | 150 | 4% | +17.2% | +34.6% | -3.1% | -4.4% | ✓ OK |
| Crisis | 손실방어 | 179 | 5% | +4.7% | +45.5% | -3.5% | -9.1% | ✓ OK |

**의도 달성: 5/5 ✓**

### 주요 발견

- **Slowdown**: 전략 DD -14.3% vs BM DD -30.2% → **벤치마크 대비 2배 이상 방어**
- **Crisis**: 전략 DD -3.5% vs BM DD -9.1% → **벤치마크 대비 2.6배 방어**
- **Goldilocks**: CAGR +12.6%로 BM +8.1% 대비 초과수익 달성

### 레짐 분포에 대한 주의사항

전체 기간의 64%가 Goldilocks로 분류됨 — 2010-2024가 미국 역사상 최장 강세장이라는 점을 감안하면 합리적이나, **Goldilocks 비중이 전체 성과를 사실상 결정하는 구조**임을 인지해야 함.  
Reflation(3%)은 표본이 너무 작아 검증 신뢰도가 낮음.

### 핵심 인사이트

1. **레짐 감지가 성과의 핵심 변수**: 3단계 결과에서 알 수 있듯, 비중의 정확한 수치보다 "언제 어떤 레짐으로 분류되는가"가 성과를 결정함.

2. **방어 레짐의 실질 역할**: Slowdown/Crisis에서 CAGR 기여는 작지만, 해당 구간 드로우다운을 벤치마크의 절반 이하로 줄이는 효과가 검증됨. 비중의 역할을 CAGR로만 평가하면 안 됨.

3. **Goldilocks 집중 리스크**: 전체 체류 64%. 레짐 감지가 실제 Goldilocks가 아닐 때 Goldilocks 비중으로 운용되면 손실이 집중될 수 있음. vol_targeting과 drawdown_thresholds가 이를 보완하는 역할.
